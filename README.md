# Customer Churn Prediction & Business Impact Analyzer

A machine learning project that predicts customer churn for a telecom company and quantifies business impact — built with a consulting lens.

## Features
- Exploratory analysis of key churn drivers (contract type, tenure, internet service)
- Random Forest classifier achieving 83.6% ROC-AUC
- Revenue impact analysis estimating ₹1.08 lakhs annual savings from model deployment
- Interactive what-if simulator to assess individual customer churn risk in real time

## Tech Stack
Python | scikit-learn | pandas | matplotlib | seaborn | ipywidgets

## Dataset
IBM Telco Customer Churn — Kaggle (7,043 customers, 21 features)
